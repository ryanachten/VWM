About VWM

Visual Working Memory is a test designed to measure individual differences in working memory. Working memory is the ability to remember information for long enough to use it. This is a core function of our complex information processing system.

This test has been created with the purpose of furthering neuropsychological assessment, investigations into working memory, and research into aspects of our cognitive architecture.

VWM requires you to remember an increasing amount of visual information, while reporting what you saw in previous scenes. As you progress through the different stages, the increasing difficulty will reflect how working memory is affected by having to hold onto an extra image.
